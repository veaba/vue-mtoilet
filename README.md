## 关于如何fix这个项目源码泄漏
- 造成build下，项目源码泄漏原因是sourceMap没关
```js
// vue cli v2.0下，src/config/index.js
module.exports={
	build:{
		productionSourceMap:false//false 即可
	}
}
// vue cli v3.0下,vue.config.js
module.exports={
	productionSourceMap:false,//这样
}
```
